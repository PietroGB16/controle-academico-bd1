Bem-vindo

Sistema de controle de atividades acadêmicas — IFRJ Campus Niterói.

Use o menu acima para gerenciar Cursos e Alunos (cadastro, edição, exclusão e busca) e para realizar consultas que cruzam as duas tabelas.

Arquitetura: o front-end (HTML + CSS + JavaScript) consome uma API REST em Node.js/Express via fetch(), que por sua vez consulta o banco de dados MySQL.
